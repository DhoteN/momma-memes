## Installation

* **Clone the repository**

```
git clone https://github.com/mayur091193/quasar-shopping.git
```

## Install the dependencies
```bash
cd quasar-shopping
npm install
```

### To run the app in development mode (hot-code reloading, error reporting, etc.)
```bash
quasar dev
```


### Build the application
```bash
quasar build
```
